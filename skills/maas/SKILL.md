# 七牛云 MaaS 平台管理助手

本 skill 帮助用户通过七牛云 MaaS API 完成以下管理任务：

- **API Key 管理**：创建、删除、启用/禁用、改名、设置日/月用量限额
- **用量与账单查询**：按月、按时间范围、按 API Key 分组查询用量和费用
- **请求日志分析**：分页查询和过滤请求日志，定位失败请求并查看详情
- **模型市场浏览**：获取模型定价、能力说明、约束参数，以及计费项名称映射

## 接口参考

所有接口定义（路径、参数、请求体、响应结构）均在 [references/openapi.json](references/openapi.json) 中，使用时请直接读取该文件获取具体细节。

## 鉴权说明

所有接口使用七牛 AK/SK 签名（`QiniuBearerAuth`）鉴权，签名算法和示例代码参见 [references/aksk-token.md](references/aksk-token.md)。

## 关键注意事项

- **API Key 完整值仅在创建时返回一次**，后续均返回脱敏版本，务必提醒用户立即保存
- **删除只能针对已禁用的 Key**，需先禁用再删除
- **禁用操作立即生效**，生产环境操作前请确认业务侧已停止使用
- `bo_usage` 字段的 key 需配合计费项接口解析为可读名称
